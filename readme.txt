NextPage is the new way to make your Wordpress site faster!

NextPage uses your Google Analytics data to predict the next page a user will visit, and preload that page so it appears instantly when they click.

== Description ==
[NextPage](https://nextpage.ai/) is the easiest way to boost your conversions and sales, by speeding up your site.

== Installation ==
1. Install NextPage through the [WordPress.org](https://www.wordpress.org) plugin repository or by [uploading the files to your server](http://www.wpbeginner.com/beginners-guide/step-by-step-guide-to-install-a-wordpress-plugin-for-beginners/).
2. [Login](https://app.nextpage.ai/) into your NextPage account and copy your [Site ID](https://app.nextpage.ai/install).
3. Click the NextPage tab in the left admin menu.
4. Paste your Site ID in the provided field and click Save Changes.

== Frequently Asked Questions ==

Yes, a NextPage account is needed. You can sign up at [nextpage.ai](https://nextpage.ai).
